# Config Backup

Мои конфигурационные файлы для Linux окружения.

## Структура

- **btop/** — настройки мониторинщика btop
- **dunst/** — конфигурация уведомлений dunst
- **fastfetch/** — настройки fastfetch
- **hypr/** — конфиги Hyprland (WM, hyprlock, keybinds)
- **kitty/** — темы и настройки терминала kitty
- **waybar/** — конфиг панели waybar
- **wofi/** — настройки лаунчера wofi
- **.zshrc** — конфиг zsh
- **.p10k.zsh** — тема powerlevel10k
